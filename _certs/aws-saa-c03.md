## About

AWS Certified Solutions Architect - Associate (SAA-C03) validates the ability to design and implement distributed systems on AWS.

## Key Domains

## Relevance to Security

Cloud security is foundational to modern offensive and defensive operations. Understanding AWS architecture enables better cloud pentesting, misconfiguration detection, and secure infrastructure design.
