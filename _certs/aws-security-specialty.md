## About

AWS Certified Security - Specialty (SCS-C02) validates advanced expertise in securing AWS workloads — identity management, infrastructure protection, data encryption, incident response, and logging/monitoring at scale.

## Key Domains

- Threat Detection and Incident Response
- Security Logging and Monitoring
- Infrastructure Security
- Identity and Access Management
- Data Protection

## Why This Cert

Pairs with the Master of Cybersecurity to bridge the gap between offensive security skills and cloud-native defence architecture. Cloud is where the money and the attack surface both live — understanding AWS security at the specialty level means being able to break it and build it.
